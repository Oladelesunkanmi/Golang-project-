# Directory Listing CLI Tool

A simple command-line tool written in Go to list all files and directories in a specified folder, along with file sizes.  

## Features

- List all files in a directory.
- List all subdirectories.
- Show file sizes in bytes.
- Easy to use with a single command.

## Installation

1. Make sure you have [Go](https://golang.org/dl/) installed.
2. Clone the repository or copy the source code.
3. Build or run the program:

```bash
go run main.go
